## Project Description
This project implements a simple rule-based chatbot using the Natural Language Toolkit (NLTK).  
The chatbot acts as a Course Assistant that responds to frequently asked questions about the AI course.

## Libraries Used
- nltk — for text tokenization and keyword matching  
- re — for regular expression-based message processing

## How to Run
1. Open `chatbot.ipynb` in Google Colab.  
2. Run all cells to initialize the chatbot.  
3. Type a question or greeting when prompted to start interacting.
